# Projeto Analise de Dados

## Contexto:
O aumento no número de clientes que cancelam cartões pode gerar perdas financeiras relevantes para instituições financeiras. Entender quem tem maior probabilidade de cancelar permite antecipar ações de retenção, melhorar a experiência do cliente e apoiar decisões estratégicas.

## Objetivo:
Explorar dados de clientes ativos e cancelados para identificar padrões de comportamento associados ao churn e desenvolver uma análise que ajude a prever clientes com maior risco de cancelamento.

## Base de dados:
Dataset com informações demográficas, relacionamento com o banco, uso do cartão e histórico de transações, incluindo tanto clientes ativos quanto clientes que já cancelaram o cartão.

Referência: https://www.kaggle.com/sakshigoyal7/credit-card-customers
